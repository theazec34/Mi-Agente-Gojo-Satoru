# 🏆 Plan Maestro - Filosofía y Estructura

> *Estructura genérica del sistema de dominio personal*

---

## 🎯 **Filosofía del Sistema**

### **Principios Infinity**
1. **Progresión Constante**: Pequeñas mejoras diarias > esfuerzos épicos
2. **Automatización Total**: Sistemas que funcionan sin tu intervención
3. **Métricas Claras**: Lo que no se mide no se mejora
4. **Feedback Loop**: Revisión y ajuste continuo

### **Componentes del Sistema**
- **Dominio de Tiempo**: Calendario automatizado
- **Dominio de Finanzas**: Presupuesto inteligente
- **Dominio de Fuerza**: Progresión estructurada
- **Dominio de Conocimiento**: Diario automatizado
- **Dominio de Hábitos**: Sistemas anti-vicios

---

## 📊 **Estructura de Progresión**

### **Ciclo 8 Semanas**
```
Semanas 1-2: Base técnica
Semanas 3-6: Progresión agresiva
Semanas 7-8: Pico de rendimiento
```

### **Métricas Clave (Ejemplo)**
```
Fuerza:
- Ejercicio A: +X% semanal
- Ejercicio B: +Y% semanal
- Ejercicio C: +Z% semanal

Finanzas:
- Deuda: -€/semana
- Ahorro: +€/semana
- Gastos: -€/semana

Salud:
- Peso: +kg/semana (masa)
- Hábitos: días/semana sin vicio
- Sueño: horas/noche
```

## 🗓️ **Estructura de Calendario**

### **Eventos Recurrentes**
```
Diarios:
- Desayuno estratégico
- Descansos anti-hábitos
- Comida principal
- Cena ligera

Semanales:
- Entrenamiento fuerza (L,M,V)
- Cardio/recuperación (Martes)
- Descanso activo (Jueves)
- Revisión semanal (Domingo)
```

### **Timing Nutricional**
```
Pre-entreno: 30min antes
Post-entreno: Inmediato
Comidas principales: Cada 3-4h
Caseína nocturna: Antes de dormir
```

---

## 🏋️ **Progresión de Fuerza**

### **Fórmula de Progresión**
```
Semana N+1 = Semana N × (1 + Incremento%)
Incremento% = 2-5% (depende del ejercicio)
```

### **Esquema de Series/Repeticiones**
```
Fase 1 (S1-2): Alto volumen, técnica
- 4x10-12 reps
- RPE 7-8

Fase 2 (S3-6): Progresión fuerza
- 4x6-8 reps
- RPE 8-9

Fase 3 (S7-8): Pico fuerza
- 3x3-5 reps
- RPE 9-10
```

---

## 💰 **Estructura Financiera**

### **Distribución de Ingresos**
```
1. Necesidades esenciales (50-60%)
2. Deuda agresiva (20-30%)
3. Ahorro emergencia (10-15%)
4. Inversión crecimiento (5-10%)
```

### **Sistema de Pagos**
```
Días 1-5: Pago deuda + necesidades
Día 15: Pago deuda + ahorro
Resto mes: Ahorro + inversión
```
