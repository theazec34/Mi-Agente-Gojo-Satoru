# 🏋️ Ejemplo: Progresión de Fuerza Estructurada

> *Estructura genérica para progresión en ejercicios compuestos*

---

## 🎯 **Principios de Progresión**

### **Ley Fundamental**
```
Progresión Semanal = Peso Actual × (1 + Incremento%)
Incremento% = 2-5% (ejercicio dependiente)
```

### **Variables de Entrenamiento**
```
1. Intensidad (% de 1RM)
2. Volumen (series × repeticiones)
3. Frecuencia (sesiones/semana)
4. Densidad (tiempo entre series)
```

---

## 📊 **Fases de Entrenamiento (8 Semanas)**

### **Fase 1: Base Técnica (Semanas 1-2)**
**Objetivo**: Dominio técnico + acondicionamiento
```
Intensidad: 60-70% 1RM
Volumen: Alto (4x10-12)
RPE: 7-8
Descanso: 60-90s
```

### **Fase 2: Progresión Agresiva (Semanas 3-6)**
**Objetivo**: Ganancia de fuerza máxima
```
Intensidad: 70-85% 1RM
Volumen: Moderado (4x6-8)
RPE: 8-9
Descanso: 120-180s
```

### **Fase 3: Pico de Fuerza (Semanas 7-8)**
**Objetivo**: Test máximo + consolidación
```
Intensidad: 85-95% 1RM
Volumen: Bajo (3x3-5)
RPE: 9-10
Descanso: 180-240s
```

---

## 📈 **Ejemplo: Press Banca Progresión**

### **Semana 1-2 (Técnica)**
```
Día 1: 4x12 @ 60% 1RM
Día 2: 4x10 @ 62.5% 1RM
Incremento semanal: +2.5%
```

### **Semana 3-6 (Fuerza)**
```
Semana 3: 4x8 @ 70% 1RM
Semana 4: 4x6 @ 75% 1RM
Semana 5: 4x5 @ 80% 1RM
Semana 6: 4x4 @ 85% 1RM
Incremento semanal: +5%
```

### **Semana 7-8 (Pico)**
```
Semana 7: 3x3 @ 90% 1RM
Semana 8: 1xMAX @ 95% + 3x2 @ 85%
Consolidación + test
```

## 🍽️ **Nutrición para Fuerza**

### **Timing Estratégico**
```
Pre-entreno (30-60min antes):
- 30g carbohidratos complejos
- 10g proteína

Post-entreno (0-30min después):
- 40g proteína rápida
- 50g carbohidratos simples

Noche (antes de dormir):
- Caseína lenta (yogur griego)
- Grasas saludables
```

### **Distribución Calórica**
```
Aumento masa:
- +500 calorías sobre mantenimiento
- 2g proteína/kg peso corporal
- 4-6g carbohidratos/kg
- 0.8-1g grasa/kg
```

## ✅ **Seguimiento y Métricas**

### **Checklist Diario Entrenamiento**
```
[ ] Ejercicio principal completado
[ ] Peso/series/repes registrados
[ ] Progresión semanal cumplida
[ ] Técnica correcta verificada
[ ] Descansos respetados
```

### **Métricas Semanales**
```
1. Peso en ejercicios principales
2. RPE promedio por sesión
3. Volumen total (series × reps × peso)
4. Tiempo total entrenamiento
5. Calidad recuperación
```

### **Indicadores de Progreso**
```
✅ Incremento peso semanal
✅ Mantenimiento/menor RPE
✅ Mejoría técnica
✅ Recuperación adecuada
❌ Estancamiento > 2 semanas
❌ Dolor articular
❌ Fatiga acumulada
```

---

## 🔄 **Ajustes del Plan**

### **Cuando Subir Peso**
```
IF todas reps completadas con RPE ≤ 8
AND técnica perfecta
THEN incrementar 2.5-5%
```

### **Cuando Mantener/Meter**
```
IF RPE ≥ 9 en últimas reps
OR técnica comprometida
THEN mantener peso siguiente semana
```

### **Cuando Bajar/Deload**
```
IF fatiga acumulada
OR lesión/dolor
OR estancamiento > 3 semanas
THEN deload 20-30% 1 semana
```

---

## 🎯 **Filosofía del Entrenamiento Inteligente**

> *"La fuerza no es solo mover peso, es la capacidad de progresar consistentemente sin romperte."*

**Principios clave**:
1. **Progresión > Intensidad**: Mejor +2.5% cada semana que +10% y lesionarte
2. **Técnica > Ego**: Forma perfecta antes de peso máximo
3. **Consistencia > Heroísmo**: 8 semanas de progreso constante > 1 día épico
4. **Escucha > Fuerza**: Tu cuerpo sabe más que tu ego
